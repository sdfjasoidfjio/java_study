자바 언어 이름 관습: 카멜 표기법
전체적으로 소문자 이름 붙임
여러 단어로 구성되는 이름은 각 단어의 첫 글자만 대문자
첫 번재 문자는 대문자로 시작

클래스 이름: 첫 번째 문자는 대문자로 시작 (ex: HelloWorld)
변수, 메소드 이름: 첫 번째 문자는 소문자로 시작 (ex: myAge, isSingle, getArea)
상수 이름: 모든 문자를 대문자로 표기(ex: PI)


자바의 데이터 타입
1.boolean 2.char 3.byte 4.short 5.int 6.long 7.float 8.double

레퍼런스 타입: 1개 이며 용도는 다음 3가지
1. 배열
2. 클래스
3. 인터페이스

문자열은 기본 타입이 아님
Stirng 클래스로 문자열 표현
문자열이 섞인 + 연산 -> 문자열 연결
ex) 
toolName + 1.8 		       -> "JDK1.8"
"(" + 3 + "," + 5 + ")"    -> "(3,5)"


상수 선언
final 키워드 사용
선언 시 초기값 지정
실행 중 값 변경 불가
ex) final double PI = 3.141592;
